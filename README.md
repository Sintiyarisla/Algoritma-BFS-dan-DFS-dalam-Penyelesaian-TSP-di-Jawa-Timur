# Algoritma-BFS-dan-DFS-dalam-Penyelesaian-TSP-di-Jawa-Timur
Secara garis besar, proyek ini juga bertujuan untuk menganalisis kinerja BFS dan DFS, baik dari segi efisiensi waktu, penggunaan memori, maupun kompleksitas algoritmik, guna memahami kelebihan dan kekurangan setiap algoritma dalam menyelesaikan TSP.
